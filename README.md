# 🍱 Gourmet Lunch Finder (ランチアプリ)

ランチタイムにぴったりのお店を検索し、気になるお店をストックできるWebアプリです。
HotPepper グルメ APIを利用しています。

## アプリの主な機能

1. **ランチ特化検索** — 場所・条件・予算で絞り込み
2. **ランチ予算の自動取得** — HotPepperの店舗ページから正確なランチ予算を自動スクレイピング
3. **ランチメニューへの直リンク** — 各店舗のランチメニューページに1クリックでアクセス
4. **ストック＆メモ機能** — 気になるお店を保存し、個別メモを残せる
5. **こだわり条件** — 個室・Wi-Fi・飲み放題などの詳細フィルター

## 🚀 GitHubで公開してスマホから使う方法（完全ガイド）

**必要なもの**: パソコン、インターネット接続、GitHubアカウント（無料）

---

### ステップ①: GitHubアカウントを作る（すでにある方はスキップ）

1. [https://github.com/](https://github.com/) にアクセス
2. 右上の **「Sign up」** をクリック
3. メールアドレス、パスワード、ユーザー名を入力して登録を完了
4. メールに届く確認メールのリンクをクリックして認証完了

---

### ステップ②: 新しいリポジトリ（保管庫）を作る

1. [GitHub](https://github.com/) にログイン
2. 画面右上の **「+」** アイコン → **「New repository」** をクリック
3. 以下のように設定:
   - **Repository name**: `lunch-app`（好きな名前でOK、半角英数字とハイフンのみ）
   - **Description**: ランチ検索アプリ（任意）
   - **Public** を選択（公開設定 — これがないとGitHub Pagesが使えません）
   - 「Add a README file」はチェック**しない**でください
4. **「Create repository」** ボタンをクリック

---

### ステップ③: アプリのファイルをアップロードする

1. リポジトリ作成直後の画面に、「**uploading an existing file**」というリンクが表示されます。それをクリック
2. パソコンのエクスプローラーで `lunch_app` フォルダを開き、以下の **4ファイル** を選択して、画面のアップロードエリアにドラッグ＆ドロップします:
   - 📄 `index.html`
   - 📄 `styles.css`
   - 📄 `app.js`
   - 📄 `README.md`（このファイル。任意）
3. 画面下部の **「Commit changes」** ボタンをクリックして保存

> ⚠️ **注意**: フォルダごとアップロードするのではなく、フォルダの**中のファイル**を直接アップロードしてください。

---

### ステップ④: GitHub Pages を有効にする

1. リポジトリのページで、上部タブの **「Settings」**（⚙️歯車アイコン）をクリック
2. 左のメニューから **「Pages」** をクリック
3. 「Build and deployment」の設定:
   - **Source**: 「**Deploy from a branch**」を選択
   - **Branch**: 「**main**」を選択 → フォルダは「**/ (root)**」のまま → **「Save」** をクリック
4. **1〜2分待ちます**

---

### ステップ⑤: 公開URLを確認する

1. 先ほどのPages設定画面をリロードすると、上部に以下のようなURLが表示されます:
   ```
   ✅ Your site is live at https://あなたのユーザー名.github.io/lunch-app/
   ```
2. このURLをクリックすると、**アプリが開きます！** 🎉

---

### ステップ⑥: スマホからアクセス & ホーム画面に追加

1. スマホのブラウザ（Safari / Chrome）で上記のURLにアクセス
2. **ホーム画面に追加する**と、アプリのように使えます：

**iPhoneの場合（Safari）:**
   - 画面下部の 📤 共有ボタン をタップ
   - 「**ホーム画面に追加**」をタップ
   - 「**追加**」をタップ

**Androidの場合（Chrome）:**
   - 右上の ⋮ メニュー をタップ
   - 「**ホーム画面に追加**」をタップ
   - 「**追加**」をタップ

---

## ⚙️ アプリ更新方法（ファイルを変更した場合）

1. GitHubのリポジトリページで、変更したいファイル（例: `app.js`）をクリック
2. 右上の **✏️ 鉛筆アイコン (Edit)** をクリック
3. 内容を編集し、**「Commit changes...」** ボタンで保存
4. 1〜2分後にGitHub Pagesに反映されます

**または**、リポジトリページの **「Add file」→「Upload files」** から最新ファイルをドラッグ＆ドロップしてもOKです（同名ファイルは上書きされます）。

## 💡 技術情報

- **API**: [HotPepper グルメ Web サービス API](https://webservice.recruit.co.jp/hotpepper/)
- **CORSプロキシ**: `corsproxy.io`（メイン）/ `allorigins.win`（フォールバック）
- **データ保存**: ブラウザの localStorage（ストック＆メモ）
- **必要なサーバー**: なし（完全な静的サイト）
